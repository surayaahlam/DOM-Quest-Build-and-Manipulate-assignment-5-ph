Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a hierarchical tree of nodes where each node
corresponds to an element, attribute, or piece of text. The DOM allows developers to access, modify, and manipulate the content, structure, and style of a webpage using
languages like JavaScript.

How do you select an element from the DOM?

You can select an element from the DOM using JavaScript methods such as:

  • document.getElementById('id'): Selects an element by its ID.
  • document.querySelector('.class' or '#id'): Selects the first element that matches a CSS selector.
  • document.querySelector('.class' or '#id'): Selects all elements that match a CSS selector.
  • document.getElementsByClassName('class'): Selects elements by class name.
  • document.getElementsByTagName('tag'): Selects elements by tag name.
What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where you attach a single event listener to a parent element instead of adding listeners to individual child elements. The event listener can detect events
that occur on its child elements using the event's bubbling mechanism (when events propagate up the DOM tree). This is useful because:

  • It improves performance by reducing the number of event listeners.
  • It makes it easier to handle dynamically added elements, as you don’t need to attach event listeners to each new child individually.
How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes and styles using JavaScript in the DOM:

  • Attributes:
    • element.setAttribute('attribute', 'value'): Sets the attribute value of an element.
    • element.getAttribute('attribute'): Gets the value of an element’s attribute.
    • element.removeAttribute('attribute'): Removes an attribute from an element.
  • Styles:
    • element.style.property = 'value': Sets inline CSS styles, e.g., element.style.backgroundColor = 'red'.
    • Alternatively, you can use element.classList.add(), element.classList.remove(), or element.classList.toggle() to manipulate classes, which apply styles from CSS rules.
